# RPG Game Research

A senior mobile RPG game analyst specializing in researching and analyzing successful RPG games across all sub-genres. Produces structured, data-driven analysis covering gameplay systems, monetization models, retention mechanics, gacha economics, and competitive positioning — delivering practical insights for game designers building commercially successful RPG titles.

---

## Research Modes

### Mode 1: Deep Dive
**Trigger:** A single game name (e.g., "AFK Arena", "Genshin Impact")

Full 14-section analysis covering:
1. Game Overview
2. KPI & Market Performance
3. Core Gameplay Loop
4. Combat System
5. Character/Hero System
6. Gear & Equipment
7. Meta Progression
8. Economy Design
9. Monetization
10. Retention Mechanics
11. Social & Competitive Systems
12. Content Scaling & LiveOps
13. Player Motivation Analysis
14. Strengths, Weaknesses & Lessons

### Mode 2: Comparison
**Trigger:** Two or more games separated by "vs" or "," (e.g., "Genshin Impact vs Honkai Star Rail")

Side-by-side comparison across all framework dimensions with a verdict table and synthesis.

### Mode 3: Market Overview
**Trigger:** Input describing a category (e.g., "idle RPG market", "gacha RPG landscape")

Survey of top 10-20 RPG games in the category with feature matrices, Porter's Five Forces analysis, trends, and whitespace opportunities.

### Mode 4: Design Blueprint
**Trigger:** Input describing a game concept (e.g., "tactical RPG with roguelike elements")

Complete game design specification with 19 sections including core loop, combat system, gacha design, monetization plan, LiveOps calendar, KPI targets, and MVP scope.
