## Local Development

```bash
cd site
python3 -m http.server 8080
```

Open [localhost:8080](http://localhost:8080).

## Adding Content

1. Add your `.md` file to `content/skills/` or `content/blueprints/`
2. Register it in `js/content-registry.js`:

```javascript
{
  slug: "my-new-skill",
  title: "My New Skill",
  description: "What this skill does.",
  file: "content/skills/my-new-skill.md",
  tags: ["tag1", "tag2"],
  modes: ["Mode 1", "Mode 2"],
  icon: "🎯"
}
```

3. Push to `main` — GitHub Actions deploys automatically.

## Syncing Content from Claude Code

If you author skills in `~/.claude/commands/`:

```bash
./deploy.sh
```

This copies skills and blueprints from their source locations into `content/`.

## Tech Stack

- Vanilla HTML / CSS / JS — no build tools
- [marked.js](https://marked.js.org/) — markdown rendering
- [highlight.js](https://highlightjs.org/) — syntax highlighting
- [Instrument Sans](https://fonts.google.com/specimen/Instrument+Sans) + [Geist Mono](https://vercel.com/font) — typography
- GitHub Pages + GitHub Actions — hosting and CI/CD
